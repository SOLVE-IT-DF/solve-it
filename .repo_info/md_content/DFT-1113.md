# DFT-1113

**ID:** DFT-1113

**Name:** Access internal storage via bootable environment

**Objective(s):** Access device data for acquisition

**Description:**

Booting the suspected computer device into forensically-sound bootable software-write blocked operating system in order to image the disks.

**Synonyms:**



**Details:**



**Subtechniques:**



**Examples:**

- WinFE - Windows Forensic Environment, is forensically sound, bootable Windows operation system that can be burned into USB and plug the USB into suspected device. Then, after changing boot order, booting to the USB to create forensic images from the tested Windows, MacOS and Linux systems.
- PALADIN Forensic - is a modified “live” Linux distribution based on Ubuntu that simplifies various forensics tasks in a forensically sound manner via the PALADIN Toolbox.
