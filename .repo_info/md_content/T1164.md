# T1164

**ID:** T1164

**Name:** Direct data read from a block device

**Objective(s):** Acquire data

**Description:**

Rather than imaging a block device, this involves accessing it directly without the expectation of saving block data to a forensic image format.

**Synonyms:**



**Details:**

This would typically be followed by processing of the data that is read to then provide a live view of partitions and file systems, and facilitates logical file extraction or any other direct operations on the data.
