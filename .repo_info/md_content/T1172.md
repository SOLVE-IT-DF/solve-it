# T1172

**ID:** T1172

**Name:** Access raw image content

**Objective(s):** Read data from digital evidence storage formats

**Description:**

This technique reads data from a raw image e.g. a dd image.

**Synonyms:**

read dd file, 

**Details:**

This provides the 'raw' equivalent of reading data from a forensic image container e.g. EWF. This provides random access to the bitstream so the stored sectors can be processed further.
