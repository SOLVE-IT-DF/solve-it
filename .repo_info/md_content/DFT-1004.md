# DFT-1004

**ID:** DFT-1004

**Name:** Selective file acquisition

**Objective(s):** Acquire data

**Description:**

Views a set of files (typically via direct access from previewing) and acquires a subset of them.

**Synonyms:**

Logical imaging, 

**Details:**

This selection of files would subsequently be stored into a container such as a tar/zip file or a forensic image format.

**Subtechniques:**



**Examples:**

- FTK Imager - allows an AD1 logical container to be created of a selected subset of files within a previewed data source.
