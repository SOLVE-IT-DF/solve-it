# DFT-1155

**ID:** DFT-1155

**Name:** Operation-based event reconstruction

**Objective(s):** Reconstruct events

**Description:**

The process of reconstructing that an operation occurred on a digital system

**Synonyms:**

Activity-based event reconstruction, Action-based event reconstruction, 

**Details:**

While some event reconstruction needs a time component, there are some questions that are simply explaining what happened, or if something happened. The time component may need to investigated later, or may be learned as a byproduct of reconstructing the event, but isn't intrinsic to the question being asked.
e.g. Was this file opened? How many times was this file opened? What data has been exfiltrated?
